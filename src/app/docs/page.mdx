# Introduction

NeuroLab is a multimodal platform for EEG analysis, clinical workflows, and AI-assisted interpretation. This documentation site is scoped to the parts of the codebase that are currently documented and maintained here.

## What is NeuroLab?

NeuroLab is designed for:

- **Mental health monitoring** through EEG-driven state analysis
- **Neurofeedback applications** that react to brain activity in near real time
- **Clinical workflows** such as appointments, billing, and session tracking
- **Research and experimentation** around neural data and model-backed analysis

## Current platform coverage

The repository currently contains:

- a Node.js backend for platform and clinical operations
- a Python AI backend for inference, reports, and chat workloads
- platform and web clients
- a docs site covering the implemented and documented surface area

## Mental states

The current documentation covers the three primary states used across the platform:

- **Relaxed**
- **Focused**
- **Stressed**

## Architecture overview

At a high level, the system is split into:

```text
Frontend clients
    ↓
Backend API (Express.js)
    ↓
AI services (FastAPI / Python)
    ↓
MongoDB, Redis, and related infrastructure
```

See [Architecture](/docs/architecture) for the more detailed breakdown.
